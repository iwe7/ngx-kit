## Demos

https://ngx-kit.com/kit/modules/spinner

## API

### `<kit-spinner>`

#### Inputs

| Property | Type | Description |
| --- | --- | --- | --- |
| *size* | `number` | Size in pixels |
| *duration* | `string` | Animation cycle duration (e.g. `'1s'`) |
| *type* | `string` | Shape (from theme) |

#### Outputs

none

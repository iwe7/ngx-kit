Ngx-kit packages

Core

Low level services and directives for hiding complex logic

Collection

Set of components based on ngx-kit core and packed into Angular-cli schematics

How it works

1. Install the packages

npm i @ngx-kit/core --save
npm i @ngx-kit/collection --save-dev

2. Import core modules

@NgModule({{ '{' }}
  ...
  imports: [
    ...
    KitForRootModule,
    KitModule,
    KitPlatformBrowserModule,

3. Generate via Angular CLI

Pick a module from Collection and put the code to your project.

ng g ui-button -c=@ngx-kit/collection [name]

4. Modify by your requirements

Update templates and styles. Only complex (but stable) logic is stored in Core.
